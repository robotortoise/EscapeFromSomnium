# EscapeFromSomnium
Codebase for the visual novel "Escape From Somnium", released to coincide with April Fool's 2024.

This project was developed in March of 2024 to test the Unity and Naninovel engines. To compile, you'll need a copy of the Naninovel and Text Animator plugins for Unity.

Source assets are archived at Archive.org.

- Jordan, lead writer and co-director.
